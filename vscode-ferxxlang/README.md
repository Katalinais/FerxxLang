# FerxxLang Syntax

Extensión de VSCode para resaltar sintaxis de archivos FerxxLang (`.fxx`).

## Instalación desde VSIX

1. Abre VSCode.
2. Ve a Extensions.
3. Abre el menú `...`.
4. Selecciona `Install from VSIX...`.
5. Escoge el archivo `ferxxlang-syntax-0.1.0.vsix`.
6. Recarga VSCode si te lo pide.

Los archivos `.fxx` se asociarán automáticamente con FerxxLang.
