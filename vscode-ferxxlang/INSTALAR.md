# Instalar FerxxLang Syntax en VSCode

Comparte el archivo:

```text
ferxxlang-syntax-0.1.0.vsix
```

Tu compañera puede instalarlo de cualquiera de estas formas:

## Desde VSCode

1. Abrir VSCode.
2. Ir a `Extensions`.
3. Abrir el menú de tres puntos `...`.
4. Elegir `Install from VSIX...`.
5. Seleccionar `ferxxlang-syntax-0.1.0.vsix`.
6. Recargar VSCode si lo pide.

## Desde terminal

```bash
code --install-extension ferxxlang-syntax-0.1.0.vsix
```

Después de instalar, los archivos `.fxx` se verán como lenguaje `FerxxLang`.
