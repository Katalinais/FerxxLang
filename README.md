# FerxxLang

Compilador para el lenguaje de scripting **FerxxLang** (extensión `.fxx`).
Construido con Flex y Bison.

## Requisitos

- `flex`
- `bison`
- `gcc`

## Compilar

```bash
make
```

## Ejecutar

```bash
./ferxxlang archivo.fxx
```

## Pruebas

```bash
make test
```

## Limpiar

```bash
make clean
```
